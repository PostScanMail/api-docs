# cURL Examples

Set env vars:
```bash
export PSM_API_KEY="YOUR_API_KEY"
export PSM_BASE_URL="https://api.postscanmail.com/api/account-docs/v2"
```

Run scripts:
```bash
bash mail-items-list.sh
bash automation-status.sh
bash automation-toggle.sh auto_scan 1
```
